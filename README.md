### A Simple React Tic-Tac-Toe one day build 


### Development
#### Getting Started  

Clone repo using

```
git clone https://github.com/roeintheglasses/apollo.git
```

#### Install yarn dependencies

cd within the repo directory

```
yarn install
```
#### Scripts 

##### Start

within the repo directory use

```
yarn start
```
to start the express server or for the react dev server use

```
yarn react-start
```

##### Build

within the repo directory use

```
yarn build
```
